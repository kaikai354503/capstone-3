# Capstone 3 – EasyShop (Description)

RecordShop is a robust, production-ready RESTful API. Built on the industry-standard Spring Boot framework, ShopEngine provides a secure, scalable backend architecture that bridges the gap between complex database management and seamless user experiences.

---

## Key Technologies

* Language: Java

* Framework: Spring Boot (REST API)

* Database: MySQL

* Security: JWT (JSON Web Tokens) for Authentication/Authorization

* Testing: Insomnia



---

## Features & Tasks

1. Bug Fixing: Identifying and resolving logic errors in the legacy Version 1 code using manual debugging and unit tests.

2. Feature Extension: Building new endpoints to enhance the shopping experience (Cart management, Order history, etc.).

3. Data Integrity: Managing complex relationships between users, products, and categories in MySQL.

4. Front-end Integration: Testing API responses against a provided JavaScript front-end to ensure a seamless UI/UX.
